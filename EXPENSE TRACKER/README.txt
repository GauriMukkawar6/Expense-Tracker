# Personal Expense Tracker

A web-based expense tracking application with user authentication and real-time dashboard.

## Features
- User Signup & Login
- Add Income & Expenses
- Transaction History
- Dashboard with analytics
- LocalStorage-based data persistence

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Open login.html in browser
2. Sign up and login
3. Start tracking expenses
